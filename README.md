# 🤫 Quiet Zone Map (소음 대피소)

> **"커피 맛보다 '몰입'이 필요한 당신을 위한 지도"**
> 카공족, 프리랜서, 개발자를 위해 소음도와 콘센트 유무를 중심으로 장소를 찾는 React 기반 웹 애플리케이션입니다.

## 1. 프로젝트 개요 (Project Overview)

기존 지도 앱들은 '맛집'이나 '별점' 위주로 정보를 제공하여, 공부나 업무를 위해 **조용한 환경**을 찾는 사용자들의 니즈를 충족시키기 어려웠습니다.
**Quiet Zone Map**은 사용자들이 실시간으로 장소의 **소음 상태**(조용함/시끄러움)를 공유하고, **콘센트 유무**를 필터링하여 최적의 몰입 환경을 찾을 수 있도록 돕습니다.

## 2. 기술 스택 (Tech Stack)

서버 없이 React의 기능을 최대한 활용하여 **클라이언트 사이드 렌더링(CSR)** 방식으로 구현했습니다.

- **Frontend:** React, Vite
- **Language:** JavaScript (ES6+)
- **Map API:** Kakao Maps SDK (`react-kakao-maps-sdk`)
- **State Management:** React `useState`, `useEffect`
- **Data Persistence:** Browser `localStorage` (서버 대용)
- **Styling:** CSS3 (Responsive Layout)

## 3. 주요 기능 (Key Features)

### 📍 1. 직관적인 소음 상태 시각화

- 지도 위의 마커 이미지를 통해 현재 장소의 상태를 즉관적으로 파악할 수 있습니다.
  - ⭐ **파란 별:** 조용함 (집중하기 좋은 상태)
  - 🔴 **빨간 핀:** 시끄러움 (소음 발생 중)

### 🔍 2. 스마트 필터링 (Smart Filtering)

- 사용자의 목적에 맞춰 원하는 장소만 지도에 남길 수 있습니다.
  - **전체 보기:** 모든 장소 표시
  - **조용한 곳만:** 현재 '조용함' 상태인 곳만 필터링
  - **콘센트 필수:** 콘센트가 설치된 장소만 필터링

### 📢 3. 사용자 참여형 상태 업데이트 (Interactive)

- 사용자가 직접 장소의 상태를 바꿀 수 있습니다.
- "시끄러워요" 버튼 클릭 시 → 마커가 즉시 **빨간색**으로 변경
- "조용해요" 버튼 클릭 시 → 마커가 즉시 **파란색**으로 변경

### 💾 4. 데이터 영속성 (Data Persistence)

- 별도의 백엔드 서버 없이 **LocalStorage**를 활용하여 데이터를 관리합니다.
- 새로고침을 하거나 브라우저를 껐다 켜도, 사용자가 업데이트한 소음 상태 정보가 사라지지 않고 유지됩니다.

## 4. 구현 내용 (Implementation Details)

본 프로젝트는 React의 핵심 개념을 충실히 반영하여 제작되었습니다.

1.  **상태 관리 (State Management):**
    - `useState`를 사용하여 장소 데이터(`places`), 필터링 조건(`filter`), 선택된 마커(`selectedId`) 등을 관리.
    - 상태 변경 시 UI가 즉시 리렌더링되어 사용자 경험(UX) 향상.
2.  **이벤트 처리 (Event Handling):**
    - 마커 클릭(`onClick`) 시 정보창 표시/숨김 처리.
    - 필터 버튼 및 상태 변경 버튼에 이벤트 리스너 연결.
3.  **데이터 모의 (Mock Data) & 라이프사이클:**
    - `useEffect`를 활용하여 앱 실행 시 로컬 스토리지의 데이터를 우선 로드하고, 없을 시 초기값(`mockData`)을 로드하는 로직 구현.
4.  **화면 설계:**
    - 헤더(컨트롤러)와 지도 영역을 분리한 레이아웃.
    - 반응형 CSS를 적용하여 윈도우 크기에 맞춰 지도 사이즈 자동 조절.
